## 1. 产品概述
网文听书试音工具是面向网络小说作者的纯前端应用，解决作者为连载小说制作有声样音时不会专业剪辑的痛点。作者只需粘贴正文，系统自动智能分段，通过可视化操作完成角色配音、参数调整，最终导出样音和配音清单，方便给读者试听或交付制作人员。

- 核心价值：降低有声化门槛，让零剪辑基础的作者也能产出专业级样音
- 目标用户：网络小说作者、IP运营方、有声书爱好者

## 2. 核心功能

### 2.1 功能模块
1. **章节导入页**：正文粘贴区、智能分段引擎、分段预览与编辑、章节管理
2. **角色配音页**：角色类型标记、声音风格选择、语速/停顿/情绪调节、未配音提示、单句试听
3. **试听导出页**：按章节连续播放、重读标记、播放进度控制、样音导出、配音清单导出

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 章节导入页 | 正文输入区 | 支持大段文本粘贴，章节标题自动识别 |
| 章节导入页 | 智能分段 | 按章节标题、对话（引号包裹）、旁白自动切分 |
| 章节导入页 | 分段预览 | 列表展示所有分段，可手动合并/拆分/编辑 |
| 章节导入页 | 章节管理 | 显示字数、分段数统计，支持多章切换 |
| 角色配音页 | 角色标记面板 | 男主/女主/反派/旁白四种快捷标记按钮 |

## 3. 核心流程

作者打开工具 → 进入章节导入页，粘贴小说正文 → 系统自动识别章节标题、对话、旁白完成智能分段 → 作者手动调整分段（合并/拆分/修改）→ 进入角色配音页 → 逐句点选标记角色类型（男主/女主/反派/旁白），页面用红色边框提示未配音句子 → 为每个角色选择声音风格（清冷/少年/沉稳/搞笑）→ 逐句调节语速、停顿、情绪强度 → 单句试听确认效果 → 进入试听导出页 → 按章节连续播放，边听边为需要重录的句子打"重读"标记 → 确认后导出样音文件和配音清单文档

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：墨韵蓝 `#1e3a5f`（沉稳文学感）作为主色，暖琥珀 `#d4a574`（书卷气息）作为强调色，米白 `#faf7f2` 作为背景
- **辅助色**：男主靛青 `#3b82f6`、女主樱粉 `#ec4899`、反派暗红 `#dc2626`、旁白墨灰 `#6b7280`（用于角色标记区分）
- **按钮风格**：圆角胶囊形按钮，主按钮带微渐变阴影，悬停有轻微上浮动画
- **字体方案**：标题用「思源宋体」体现文学质感，正文用「思源黑体」保证可读性
- **布局风格**：顶部固定导航栏（三页切换+进度指示），主体采用卡片式分层布局，左侧操作区右侧预览区
- **装饰元素**：仿宣纸纹理背景、细墨线分隔符、古雅书卷图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI细节 |
|-----------|-------------|-------------|
| 章节导入页 | 正文输入区 | 仿稿纸风格大文本框，带行号和字数实时统计 |
| 章节导入页 | 分段预览列表 | 段落卡片，不同类型（标题/对话/旁白）用左侧色条区分，卡片可拖拽排序 |
| 角色配音页 | 句子列表 | 每行一条句子，左侧角色标记色块（未标记为红色虚线），右侧参数快捷入口 |
| 角色配音页 | 参数调节面板 | 滑动条带刻度，滑块拖动实时更新数值气泡，情绪强度有表情图标辅助 |
| 角色配音页 | 声音风格卡片 | 卡片式风格选择，选中有琥珀色边框，带风格描述文字 |
| 试听导出页 | 播放器 | 黑胶唱片风格进度条，播放/暂停/上一句/下一句控制，当前句子高亮滚动 |
| 试听导出页 | 重读标记 | 每行句子右侧星标按钮，点亮后整行背景微变琥珀色 |
| 试听导出页 | 导出区 | 大按钮带文件图标，导出时显示进度条和完成动画 |

### 4.3 响应式设计
- 桌面优先（≥1280px）：双栏布局，左操作右预览，充分利用宽屏
- 平板端（768px-1279px）：折叠为单栏，操作区与预览区上下排列
- 移动端（<768px）：导航改为底部Tab，句子列表全屏滚动，参数面板改为底部抽屉弹出
- 所有触控目标≥44x44px，支持滑动操作
